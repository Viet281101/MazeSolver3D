# 3D Maze Solving

![alt text](src/typescript.svg) & Threejs, dat.gui

### Compile & Debug

- Use `npm` command:

```bash
npm install
npm run dev
```

### Deploy project to Github Pages

- Use `npm run` to build and test folder `dist/`:
```bash
npm run build
npm run preview
```

- Deploy to Github Pages:
```bash
npm run deploy
```
